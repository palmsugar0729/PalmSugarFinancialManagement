# 卡片化设计风格

持仓列表中的产品信息卡设计规范：

- 每个理财/投资产品使用独立 `Card` 展示，圆角 12px
- 卡片底色使用主色调 `#A3C1AD` withAlpha(15) 做柔和背景
- 卡片内容：基金名称（加粗）+ 代码标签 + 类型 → 市值/成本/盈亏三栏 → 份额/成本价/净值 → 收益率进度条
- 盈亏正数红色、负数绿色（红涨绿跌）
- 点击卡片进入详情页，详情页底部有快捷操作按钮（买入/卖出/分红）

参考文件：[[investment_page.dart]]

**Why:** 用户偏好卡片化布局，后续做其他产品（股票、期货）时复用此设计
**How to apply:** 新增产品持仓卡片时参照此规范，保持底色、圆角、字段排版一致
