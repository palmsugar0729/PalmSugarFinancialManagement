## 话题 5：第一期 MVP 代码开发

### 5.1 数据库层

| 文件 | 说明 |
|------|------|
| `lib/database/models.dart` | Category、Transaction、Answer 数据模型 |

### 5.2 页面层

| 文件 | 功能 |
|------|------|
| `lib/pages/home_page.dart` | 首页：当月收支汇总卡片 + 最近记录列表，支持下拉刷新、左滑删除、点击编辑 |
| `lib/pages/add_record_page.dart` | 记一笔：类型切换（支出/收入/转账）、金额输入、日期选择、分类选择、备注，支持新增/编辑 |
| `lib/pages/category_page.dart` | 分类管理：Tab 切换三类，展示预设分类，支持自定义分类增删改 |
| `lib/pages/answer_book_page.dart` | 答案之书：提问 → 随机抽取答案，带分类标签和渐变卡片效果 |

### 5.3 主题与入口

- `lib/main.dart`：Material 3 + Teal 主色调，跟随系统亮暗色

### 5.4 遇到的问题

1. **CardTheme → CardThemeData**：Flutter 3.44 中 `cardTheme` 参数类型变更为 `CardThemeData`
2. **Transaction 命名冲突**：`sqflite` 库中也有 `Transaction` 类，通过 `hide Transaction` 解决
