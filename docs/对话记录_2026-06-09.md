# 2026-06-09 对话记录

## 主题

Excel 导入导出功能开发 + 月份切换 + 分类管理增强 + 多项 UI/交互优化

## 关键决策

| # | 决策 | 原因 |
|---|---|---|
| 1 | 使用 `excel` 包读写 xlsx | 纯 Dart 实现，无需原生依赖 |
| 2 | `share_plus` 改为用户主动触发 | 模拟器/部分设备无分享应用，自动调用会弹「没有应用可执行此操作」 |
| 3 | `excel.rename()` 替代 `delete()` | `excel` 包 v4.0.6 中 `delete()` 不生效 |
| 4 | 去掉分类拖动排序 | 用户反馈不好看，且不影响使用 |
| 5 | 月度总览颜色：收入=红、支出=绿 | 与理财习惯一致（红涨绿跌） |
| 6 | 使用 `flutter_slidable` 实现左滑删除 | 可以露出红色删除按钮，无需确认对话框 |
