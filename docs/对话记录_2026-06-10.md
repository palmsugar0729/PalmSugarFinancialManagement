# 对话记录 — 2026-06-10

## 主题

收支分析图表页面开发 + 支付宝/微信导入初步讨论

## 关键决策

1. **图表库选择**：使用 `fl_chart` v0.68.0（Flutter 社区最活跃的图表库）
2. **入口设计**：首页每月总览卡片点击后进入分析页面；月份选择功能移到 AppBar 日历图标
3. **页面结构**：Tab 切换 — 趋势 / 支出 / 收入
4. **支付宝/微信导入**：确认无公开 API，只能走「用户导出文件 → App 导入」方式；用户自行研究 CSV 格式
