# 需求 v1.0（已封存 · 2026-06-13）

> 本文件为原始需求，大部分已完成。未完成项已迁移至 [need_2.0.md](need_2.0.md)。

一个财务管理APP，可以记录自己的收支、可以导入主流应用的账单、可以记录理财交易内容。
先做安卓端、再做苹果端。
## 核心功能
- 手动输入收入和支出，能选择类型
- 可以导入账单文件，并自动根据信息进行录入
- 可以导入主流支付应用的账单文件，例如支付宝和微信，并自动根据信息进行录入
- 可以手动记录理财交易内容，也可以导入理财app的信息，例如同花顺等。
- 可以导出excel作为存档。
## 其他功能
- 尽可能轻量化
- 答案之书功能，即问一个问题，回答一句话。类似于占卜。回答的内容可能先关，可能完全不相关。可以查找答案之书相关内容了解
- 收支分析
- 投资交易分析
