## 内容摘要

### 1. MuMu 模拟器配置

- 配置 ADB 连接：`adb connect 127.0.0.1:7555`
- App 成功运行到 MuMu 模拟器（Android 12）
- MuMu 模拟器无法弹出软键盘（模拟器限制）

### 2. 分类管理「添加分类」Dialog 闪退 Bug

**现象**：添加分类时无论输入/取消/点空白都闪退，红屏报错 `_dependents.isEmpty`。

**根因**：`TextEditingController` 在 dialog 关闭动画未完成时手动 `dispose()`。

**修复**：controller 移入 dialog `builder` 内部，随 dialog 自动销毁。

**附加优化**：Hint 文本按类型动态变化（支出/收入/转账显示不同示例）。

### 3. 真机安装 APK 闪退 Bug

**现象**：真机安装后点击图标直接闪退，`Could not find 'libflutter.so'`。

**根因**：APK 只包含 `x86_64` 库，真机需要 `arm64-v8a`。

**修复**：`flutter build apk --debug --target-platform=android-arm64`

**后续**：abiFilters 配置 + 清理旧 APK。

### 4. 金额输入计算器功能

**需求**：金额输入框支持简单加减乘除，如 `35.43+21.82` 按回车自动计算结果。

**实现**：`onEditingComplete` + `_evaluateExpression()` 正则解析表达式。

**衍生 Bug**：`InputFormatter` 正则只匹配单个字符，导致只能输入一位数字。

**修复**：`RegExp(r'^[\d+\-*/.]*$')` 匹配任意长度。

### 5. 键盘焦点修复

**问题**：按回车/点击空白处不脱离输入状态，键盘不收起。

**修复**：`onEditingComplete` 中 `FocusScope.of(context).unfocus()` + Body 包 `GestureDetector.onTap`。

### 6. UI 微调

- 「记一笔」保存按钮：`TextButton` → `FilledButton`（更明显）

### 7. 真机测试最终修正（用户截图反馈）

**时间显示 `00:00`**
- 现象：最近记录列表中时间均为 `00:00`
- 根因：`_saveTransaction` 中 `DateTime` 只传年月日，时分秒默认 0
- 修复：加入 `DateTime.now()` 的时、分、秒

**收入/支出颜色反了**
- 现象：收入绿、支出红，与「红涨绿跌」习惯相反
- 修复：三个页面 `_getTypeColor` 统一改为收入=红、支出=绿

**日期选择器显示英文**
- 现象：`Select date` / `Cancel` / `OK` 等均为英文
- 修复：`MaterialApp` 添加 `flutter_localizations` + `locale: zh_CN`
- 附加：`intl` 升级 `0.19.0` → `0.20.2` 解决依赖冲突

**App 改名**
- `PalmSugar 记账` → `棕榈糖账本`
- 涉及：`main.dart` title、`home_page.dart` AppBar、README.md
