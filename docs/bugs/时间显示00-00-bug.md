# 最近记录时间显示为 `00:00`

**状态**：🟢 已解决
**发现日期**：2026-06-08
**关联**：[[2026-06-08_开发日志]] · [[add_record_page.dart]]

---

## 现象

首页「最近记录」列表中，所有记录的时间部分均显示为 `00:00`，日期部分正确。

---

## 根因

保存交易记录时，`_saveTransaction` 中构造 `DateTime` 只传递了年月日：

```dart
final dateMs = DateTime(
  _selectedDate.year,
  _selectedDate.month,
  _selectedDate.day,
).millisecondsSinceEpoch;
```

未传递时、分、秒，默认为 `0`，导致数据库中存储的时间戳对应 `00:00:00`。

---

## 解决方案

保存时加入当前时间的时、分、秒：

```dart
final now = DateTime.now();
final dateMs = DateTime(
  _selectedDate.year,
  _selectedDate.month,
  _selectedDate.day,
  now.hour,
  now.minute,
  now.second,
).millisecondsSinceEpoch;
```
