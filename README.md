# 棕榈糖账本

个人财务管理 Flutter App — 记录收支、管理理财、导入导出、图表分析。该 App 通过 Vibecoding 开发。

## 技术栈

- **Flutter** — 跨平台移动开发框架（Material 3）
- **SQLite** — 纯本地数据存储（sqflite）
- **fl_chart** — 图表
- **excel / csv** — 导入导出
- **flutter_slidable** — 左滑删除

## 功能概览

### 首页（记账）
- 手动收支录入（金额计算器、日期选择、分类选择、备注）
- 月度收支汇总卡片 → 点击进入图表分析
- 当月交易列表：左滑删除、长按多选批量删除
- 月份切换（选择器 + 左右滑动）
- 数据备份入口（导入导出）

### 分析
- 月度收支趋势柱状图
- 支出/收入分类饼图
- 点击分类查看当月交易明细

### 理财
- 基金持仓列表（市值居中大字 + 成本/收益/收益率）
- 点击持仓卡片 → 基金详情（净值更新 + 快捷买入/卖出/分红）
- 交易记录：左滑删除、长按多选批量删除
- 基金管理（新增/编辑/删除基金）
- 投资分析（总览 + 持仓占比饼图 + 基金市值对比柱状图）
- 买入/卖出/分红录入（手续费 + 净值 + 份额 → 自动算金额）
- 基金代码联网查名称（天天基金接口）
- 自动同步到首页：买入→transfer"投资"、卖出→transfer"赎回"、手续费→expense、分红→income"投资收益"
- 成本采用移动加权平均法

### 答案之书
- 趣味占卜，自定义词条（`answers_template.json`）

### 导入导出
- Excel：一个文件三个 Sheet（收支记录 + 理财记录 + 分类表）
- CSV：UTF-8 BOM 编码，包含收支和理财两段
- 导出选项：选择内容（全部/收支/理财）+ 时间范围（近1月/近3月/自定义）
- 导入自动识别格式，不存在的基金自动创建，重复分类自动跳过

## 使用说明

### 记账

底部 **首页** Tab → 右下角 **+** 按钮进入记一笔。

- 选择类型（支出/收入/转账）→ 输金额（支持表达式如 `10+5`）→ 选日期 → 选分类 → 写备注 → 保存
- 金额输入框支持简单四则运算，如 `30*2` 自动计算为 60
- 月度汇总卡片点击进入**图表分析**，可看趋势图和分类饼图
- 左滑任意记录可删除，长按进入**多选批量删除**模式

### 理财

底部 **理财** Tab 进入投资管理。三个核心操作：

**1. 新建基金**：右上角齿轮 → 基金管理 → 右下角 + → 填写名称/代码（可选）/类型 → 添加。
也可输入 6 位代码点「查询」联网获取基金名称。

**2. 录入交易**：右下角 + → 选择买入/卖出/分红。

| 字段 | 说明 |
|------|------|
| 选择基金 | 下拉选已有基金，或点「新建基金」 |
| 日期 | 交易日期 |
| 手续费 | 选填，平台收取的费用 |
| 净值 | 基金单位净值 |
| 份额 | 买入/卖出的份额数 |
| 金额 | **自动计算**（份额 × 净值），无需手动填 |
| 备注 | 选填 |

- **买入**：金额 = 份额 × 净值，手续费不影响持仓成本
- **卖出**：份额不能超过当前持仓，金额 = 份额 × 净值
- **分红**：只需填金额，选基金即可

**3. 查看持仓**：持仓 Tab 展示所有基金的卡片。

- 卡片中央大字为**持仓金额**（= 份额 × 最新净值）
- 下方依次：持仓成本 \| 持有收益 \| 收益率
- 点击卡片 → 基金详情页：可更新净值、查看交易记录、快捷买入/卖出/分红
- 净值需要手动更新（在详情页输入最新净值点「更新」）

**自动同步到首页**：
| 理财操作 | 首页记录 |
|---------|---------|
| 买入 | transfer「投资」 |
| 买入手续费 | expense「手续费」 |
| 卖出 | transfer「赎回」 |
| 卖出手续费 | expense「手续费」 |
| 分红 | income「投资收益」 |

**批量删除**：交易记录 Tab → 长按任意一条 → 全选 → 删除。

### 导入导出

首页右上角备份图标 → 数据备份。

- **导出**：选内容（全部/收支/理财）→ 选时间范围（近1月/近3月/自定义）→ 选格式（Excel/CSV）
- Excel 文件包含三个 Sheet：收支记录、理财记录、分类表
- CSV 文件 UTF-8 编码，可直接用 Excel/WPS 打开
- **导入**：选文件 → 系统自动识别格式和数据 → 不存在的基金会自动创建 → 重复分类自动跳过

### 答案之书

底部 **答案之书** Tab。点击屏幕获得一句随机回答。自定义词条：编辑根目录 `answers_template.json`，填 `cn`（中文）和 `en`（英文）字段。
